# FIFA-World-Cup-2026-Dashboard-

# Objectif du projet :
Ce projet vise à analyser les performances des joueurs participant à la Coupe du Monde FIFA 2026 à l'aide de Power BI. L'objectif est d'identifier les joueurs les plus performants, les équipes les plus efficaces et les tendances de performance à travers différents indicateurs clés.
# Dataset:
Dataset : FIFA World Cup 2026 Player Performance Dataset
Nombre de lignes : 50 000+
Domaine : Football / Sport Analytics
# Outils utilisés:
Power BI Desktop
Power Query
DAX
Microsoft Excel
# Etapes de traitement des données:
# 1.Importation des données:
Chargement du dataset dans Power BI Desktop.
Vérification de la structure des données.
# 2. Nettoyage des données avec Power Query:
Vérification des types de données.
Suppression des colonnes inutiles.
Gestion des valeurs manquantes.
Conversion des formats numériques.
Création de la colonne :
goal_contribution = goals + assists
# 3. Modélisation des données :
Création d'une table Calendar.
Création des relations entre les tables.
Ajout des colonnes Year et Month.
# 4. Création des mesures DAX:
Total Goals
Total Assists
Average Rating
Average Performance
# 5. Création du Dashboard:
KPI Cards
Graphiques en barres
Graphiques circulaires
Graphiques de tendance
Segments de filtrage (Slicers)
 KPI Principaux:
Total Goals
Total Assists
Average Rating
Average Performance Score
# Visualisations réalisées :
Top 10 Players by Goal Contribution
Goals by Team
Performance by Position
Average Performance Over Time
Player Position Distribution
# Insights obtenus:
Les attaquants présentent la plus forte contribution offensive.
Certaines équipes dominent le classement des buts marqués.
Les performances moyennes varient selon les nationalités.
Les meilleurs joueurs se distinguent par une forte contribution offensive
